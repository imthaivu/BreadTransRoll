This is Daily Speaking English: a role-based Next.js + Firebase app for students, parents, teachers, and admin.

## Quick start

- Copy `.env.local.example` to `.env.local` and fill Firebase values
- Install deps: `npm install`
- Start dev: `npm run dev`

## Roles

- student: upload audio/video, view feedback, leaderboard, update profile
- parent: track student progress
- teacher: annotate feedback (speed, intonation, pronunciation, fluency), set scores
- admin: manage users, classes, roles

## Tech

- Next.js (App Router, TypeScript, Tailwind)
- Firebase (Auth, Firestore, Storage)
